## How to Play

1. Run the Python script `rps_game.py`.
2. Enter your choice:
   - Type `0` for Rock
   - Type `1` for Paper
   - Type `2` for Scissors

3. The computer will randomly choose its move.
4. The winner will be determined based on the game rules:
   - Rock crushes Scissors
   - Scissors cuts Paper
   - Paper covers Rock
